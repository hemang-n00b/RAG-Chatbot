What does it do?

The chatbot serves as a virtual assistant for individuals seeking information and assistance regarding the coronavirus pandemic. It can answer questions about COVID-19 symptoms, transmission, and precautions. Additionally, it provides updates on local guidelines and regulations, helping users stay informed about the evolving situation. The chatbot also offers resources for mental health support and coping strategies during the pandemic.
